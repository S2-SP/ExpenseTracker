# Expense Tracker

A full-stack application to track personal expenses, built with .NET 8 (C#) for the backend, React for the frontend, and Azure Pipelines for CI/CD.

This app allows users to manage their daily, monthly, annual, or travel expenses, providing a simple interface to add, view, edit, and delete expense records.

Features

Track expenses by type: Daily, Monthly, Annual, Travel

Add, view, update, and delete expenses

Categorize expenses (Food, Transport, Bills, Travel, etc.)

Simple and responsive React frontend
